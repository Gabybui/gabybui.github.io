This repository contains Python code for generating additional plots for the assignment 1.

There are 5 plots in total: Data size changed over years, top 10 most popular elements, top 10 countries with largest number of stations, top 10 states with highest number of stations, proportion of stations in southern hemisphere

To run the code, simply open the 'additional_plot.py' file in an IDE, install all necessary packages and click the 'Run' button. The resulting plots will be displayed in separate windows.
